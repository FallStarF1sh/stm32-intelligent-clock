# STM32智能电子钟

基于 `STM32F103C8T6` 的智能电子钟项目，使用 `CubeMX + Keil` 开发，并在 `Proteus` 中完成仿真验证。

## 项目简介

本项目实现一个具备时间显示、按键设置、蜂鸣提示等功能的电子钟系统。  
系统采用 `STM32F103C8T6` 作为主控，`LCD1602` 作为显示器件，`DS3231` 作为实时时钟模块。

## 功能说明

- 实时显示时、分、秒
- 按键设置时间
- 支持蜂鸣器提示
- LCD 显示当前时间和状态
- Proteus 仿真验证

## 硬件方案

- 主控芯片：STM32F103C8T6
- 显示模块：LCD1602
- 时钟模块：DS3231
- 输入模块：4个按键
- 提示模块：蜂鸣器
